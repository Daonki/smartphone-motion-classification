# Smartphone Motion Classification (스마트폰 센서 기반 행동 인식)

📌 프로젝트 소개  
스마트폰 센서 데이터를 활용하여  
사용자의 행동을 **정적 / 동적 행동으로 분류**하는  
머신러닝 기반 행동 인식 모델을 설계·분석한 프로젝트입니다.

데이터 탐색부터 전처리, 모델링, 성능 평가, 결과 해석까지  
머신러닝 프로젝트의 전체 흐름을 단계적으로 경험했습니다.

---

🛠 사용 기술
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

---

⚙️ 주요 기능
- 스마트폰 센서 데이터 탐색적 분석(EDA)
- 행동 분류 모델 설계 및 학습
- 단계별 모델링을 통한 성능 비교
- 시각화를 통한 결과 해석

---

👩‍💻 담당 역할
- 데이터 분석 및 모델 설계
- 학습/검증 데이터 구성
- 모델 성능 평가 및 결과 해석
- 시각화 기반 분석 수행

---

📈 결과
- 정적/동적 행동 분류 모델 구축
- 모델 구성 및 데이터 관리 방식에 따른 성능 차이 확인
- 시각화를 통한 모델 예측 결과 해석

---

📂 프로젝트 구성
- `1.탐색적_데이터_분석.ipynb` : 센서 데이터 EDA
- `2.기본_모델링.ipynb` : 기본 행동 분류 모델 학습
- `3.단계별_모델링.ipynb` : 단계별 모델 개선 및 비교
