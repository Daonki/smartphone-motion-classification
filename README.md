## **스마트폰 센서 기반 데이터로 행동 인식 모델 개발**

**2025.10.15 ~ 2025.10.17**

#Python #MachineLearning #DataAnalysis

---

### 프로젝트 개요

스마트폰 센서 데이터를 활용하여 사용자 행동(정적/동적)을 분류하는 머신러닝 모델을 설계하고 분석한 프로젝트입니다. 데이터 전처리부터 모델링, 평가, 결과 해석까지 전체 개발 과정을 경험했습니다.

---

### **내가 맡은 역할**

- 데이터 분석 및 모델 설계
- 학습/검증 데이터 구성 및 모델 성능 평가
- 시각화를 활용한 결과 해석

---

### **사용 기술**

- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

---

### **경험 및 배운 점**

### 1) **모델·데이터 관리 체계의 중요성 절실히 깨달음**

- Activity 모델과 is_dynamic 모델의 이름을 명확하게 구분하지 않아
    
    병합 과정에서 오류 발생 → 학습/검증 데이터가 뒤섞이는 문제로 이어짐
    
- 단순히 모델명 수정만 하면 되는 줄 알았으나,
    
    데이터까지 잘못 섞이면 성능 수치가 완전히 왜곡될 수 있다는 것을 경험
    
- 이를 통해 “데이터·모델 버전 관리가 ML 프로젝트의 핵심”이라는 점을 체감함

### 2) **결과 해석이 모델링보다 어렵다는 것을 배움**

- 모델을 만드는 것보다, 결과를 해석하고 설명하는 과정이 훨씬 더 어렵다는 걸 느꼈음
- 특히 시각화된 데이터를 보더라도
    
    “이 데이터가 어떤 의미를 가지는가?”를 직관적으로 읽어내기 힘들었던 경험이 있음
    
- 이 경험을 통해
    
    **모델링 → 시각화 → 결과 해석**까지 연결되는
    
    “분석가로서의 사고력”이 중요함을 깨달음
    

### 3) **앞으로의 방향성**

- 모델 결과를 정확히 해석하고 의미 있게 전달하는 역량을 강화할 계획
- 특히 시각화 기반 분석 능력을 향상하여 데이터 기반 문제 해결력을 높이는 것을 목표로 함
